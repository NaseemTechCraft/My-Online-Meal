# My Online Meal

## Overview

My Online Meal is a responsive fast food restaurant website designed to showcase menu items, services, and contact options with a clean modern interface. It serves as a multi-page restaurant landing experience with menu browsing, ordering prompts, login/register navigation, and service details.

## Features

List all major features of the project.

- Responsive multi-page restaurant website
- Full navigation across menu, services, contact, and blog pages
- Order page layout for meal selection
- Login and registration page structure
- Interactive buttons and modern UI styling

## Technologies Used

List all technologies, frameworks, libraries, and tools used in the project.

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Install required dependencies. (No dependencies required for this static HTML/CSS/JS project.)
4. Run the application by opening `Index.html` in a web browser.

## Usage

Open the site in a browser to browse the restaurant pages. Use the main navigation to access the home page, menu pages, services, contact information, and login/register views. The Order page demonstrates how customers can view available meal options.
